Welcome to the Lab 2 homepage!

The purpose of this web programming lab is to help students develop a foundational understanding of how to structure and organize web pages using HTML. By creating a basic HTML page, this lab focuses on teaching essential skills such as setting up web page elements, formatting text, and incorporating both internal and external links into a webpage. Students will explore the importance of structuring content with tables. They will also learn how to use relative and absolute paths effectively to navigate between web pages. This lab provides a practical introduction to fundamental web development concepts.
